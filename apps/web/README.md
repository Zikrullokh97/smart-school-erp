# Web App

The web app contains the Next.js dashboard for platform administrators, school administrators, teachers, and support staff.

## Planned Responsibilities

- Tenant-aware admin dashboard.
- School, user, role, student, teacher, parent, schedule, attendance, notification, and reporting workflows.
- Generated API client usage.
- Real-time WebSocket notifications.
- Accessibility-conscious, responsive UI built with TypeScript, TailwindCSS, and shadcn/ui.
